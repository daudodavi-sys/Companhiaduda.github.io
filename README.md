# 📱 Companhia Duda

Plataforma especializada na **venda e recuperação de números de telefone** de todas as operadoras do Brasil. Solução rápida, segura e descomplicada para quem precisa de um novo número ou deseja reaver um número antigo.

## 🎯 Funcionalidades

- ✅ Venda de números pré-pagos e pós-pagos (Vivo, Claro, Tim, Oi)
- ✅ Recuperação de números expirados ou cancelados
- ✅ Consulta de disponibilidade por DDD
- ✅ Portabilidade facilitada
- ✅ Atendimento 100% digital

## 🛠️ Tecnologias utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (ou React, se aplicável)
- **Backend:** Node.js / PHP / Python (ajuste conforme sua stack)
- **Banco de dados:** MySQL / PostgreSQL
- **APIs integradas:** Operadoras (Vivo, Claro, Tim, Oi)

## 📦 Como instalar e rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/companhia-duda.git

# Acesse a pasta do projeto
cd companhia-duda

# Instale as dependências (exemplo com npm)
npm install

# Configure as variáveis de ambiente (arquivo .env)
# - Credenciais das APIs das operadoras
# - Chave de banco de dados

# Execute o projeto
npm start
